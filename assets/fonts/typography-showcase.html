Inter

SECONDARY · BODY TEXT & UI

Inter is an optimized typeface designed for computer screens. It features a tall x-height for excellent legibility at small sizes, plus exceptional clarity across all weights. Perfect for body text, captions, and interface elements.

Why Inter? It maintains readability even in challenging conditions—low resolution, poor lighting, or motion. The letterforms are carefully crafted to reduce eye strain during long reading sessions.

Typography Rules

BEST PRACTICES

✅ DO

  • • Use Space Grotesk for all headings
  • • Keep body text at 16-18px minimum
  • • Maintain 1.6-1.8 line height for readability
  • • Use JetBrains Mono for code only
  • • Limit to 60-80 characters per line

❌ DON'T

  • • Mix more than 3 font families
  • • Use script fonts for body text
  • • Set text smaller than 14px
  • • Use all caps for long text
  • • Stretch or distort letterforms
